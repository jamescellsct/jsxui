# Layout

Layout components manage how elements are arranged throughout an interface. JSXUI ships with the following layout components:

- Grid
- Column
- Row
- Space

## Grid

Grid arranges immediate children along predefined columns and rows.

```jsx
function App() {
  return (
    <Grid>
      <Column>
        <Row>
          <Text>Hello</Text>
        </Row>
      </Column>
    </Grid>
  )
}
```

## Column

The `Column` component arranges immediate children along the vertical axis.

```jsx
function App() {
  return (
    <Grid>
      <Column>
        <Text>Hello</Text>
      </Column>
      <Column>
        <Text>World</Text>
      </Column>
    </Grid>
  )
}
```

## Row

The `Row` component arranges immediate children along the horizontal axis.

```jsx
function App() {
  return (
    <Grid>
      <Row>
        <Text>Hello</Text>
      </Row>
      <Row>
        <Text>World</Text>
      </Row>
    </Grid>
  )
}
```

## Space

The `Space` component defines the space between children along the horizontal or vertical axis.

```jsx
function App() {
  return (
    <Grid>
      <Row>
        <Text>Hello</Text>
        <Space />
        <Text>World</Text>
      </Row>
    </Grid>
  )
}
```

### Calendar

```jsx
// https://dribbble.com/shots/14750803-Paligo-Elements/attachments/6453967?mode=media
function CalendarApp() {
  return (
    <Grid columns={daysInView}>
      {/* Create columns of days */}
      {days.map(({ dayOfWeek, date, selected }, index) => (
        <Column column={index} background={[selected, 'blue']}>
          <Text>{date}</Text>
          <Text>{dayOfWeek}</Text>
        </Column>
      ))}

      {/* Place events along those days */}
      {events.map(({ title, description, start, end, color }) => (
        <Row column={[start, end]} background={color}>
          <Text fontSize={16} fontWeight={500}>
            {title}
          </Text>
          <Text fontSize={12}>{description}</Text>
        </Row>
      ))}
    </Grid>
  )
}
```
